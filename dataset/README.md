# Lease Evaluation Dataset

This folder contains the anonymized lease agreement dataset used for evaluation in the research paper.

## Contents
- Annotated lease agreements (English and Kannada)
- Clause severity labels (High, Medium, Low)
- Predefined clause categories

## Validation
The dataset annotations were reviewed and validated by three practicing legal professionals.

## Privacy
All personally identifiable information has been removed before publication.

## Usage
This dataset is released strictly for academic and non-commercial research purposes.
